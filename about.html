About Me

My name is Kason Heilman. I am a student KSU majoring in IT. This project, for my IT 3203 class explores the history of HTML and its role in web development.

I am interested in web design and technology, and this project helped me better understand how websites are structured and how web standards evolved over time. I was first introduced to web development way back in high school in my Web Design class, but never got much of a chance to expand on my knowledge of it then. This project allows me to not only hone my skills of it, but also helps me understand the history of it, which is always important.

About This Project

This website was created as a class project using only HTML and CSS for Milestone 1. Its purpose is to present research in a clear and organized multi-page website.
