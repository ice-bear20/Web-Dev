Blurb on HTML

HTML, or HyperText Markup Language, is the standard language used to structure content on the web. Since its invention in the early 1990s, HTML has evolved from a simple document-markup system into the foundation of modern websites that we all know today.

This website explores the history of HTML, its major versions, key concepts, how it has evolved since its initial release, and its continuing role in web development.

Project Overview

About this project

This project examines the development of HTML from its earliest versions to HTML5. It explains why HTML was created, how it changed over time, and why it remains essential for organizing and presenting web content.

Key Terms

  • HTML - Hyper Text Mark up Language, the code used to make and develop websites
  • Tags - pieces of HTML code inside brackets that tell the browser what something is
  • Elements - the full parts of a webpage created with tags, such as a heading, paragraph, or link.
  • Attributes - extra details added inside a tag to give an element more information or change how it works.
  • Semantic markup - HTML that uses clear, meaningful tags to show what each part of the page is for
